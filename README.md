This repo contains my final Python project from when I completed CS50 in December 2022. 
I continued building the web app into a larger scale tutoring system for my high school.
The app was hosted on Heroku with a PostgreSQL database using Flask.
A separate README within the tutoring folder describes the web app and its pages in greater detail.
After gaining 100+ users, the app was taken offline when I graduated high school in 2024.
